CrazyFamily Doom — Lizenz- und Quellenhinweise
================================================

Zwei klar getrennte, frei lizenzierte Bausteine:

1) ENGINE  ·  doomgeneric.js + doomgeneric.wasm
   --------------------------------------------------------------------
   WebAssembly-Build der quelloffenen Doom-Engine "doomgeneric"
   (beruht auf dem 1997 von id Software unter der GNU General Public
   License freigegebenen Doom-Quellcode). Reine Single-Player-Engine,
   mit Bild + SOUND (Soundeffekte) über SDL2. Im Browser stabil und
   ohne Grafikfehler.

   Lizenz : GNU General Public License, Version 2 (GPL-2.0)
   Quelle : https://github.com/ozkl/doomgeneric
   GPL-Lizenztext: https://www.gnu.org/licenses/old-licenses/gpl-2.0.txt

   Build: mit Emscripten (emcc) aus dem doomgeneric-Quellcode, Backend
   "doomgeneric_emscripten.c", Ziel Makefile.emscripten. Angepasst wurden
   die Build-Flags (SDL2-Ports statt sdl2-config, WAD wird zur Laufzeit
   geladen statt eingebacken, -O2 -fno-strict-aliasing -fwrapv).

   MODIFIKATION (GPL-2.0 §2): An doomgeneric.c wurde ein kleiner Block mit
   reinen LESE-Funktionen angehängt (cf_kills, cf_map, cf_episode, cf_health,
   cf_pstate, cf_gamestate, cf_usergame u.a., je EMSCRIPTEN_KEEPALIVE). Sie
   geben Doom-interne Werte (Killcount, aktuelles Level, Spielerzustand) an
   JavaScript zurück, damit die Webseite einen Highscore bilden kann. Die
   Spiellogik selbst ist UNVERÄNDERT. Der vollständige Patch + die genauen
   Build-Befehle stehen in BUILD.txt (entspricht der Quelltext-Pflicht der
   GPL-2.0). Quelle der Engine unverändert: https://github.com/ozkl/doomgeneric

2) SPIELDATEN  ·  freedoom1.wad / freedoom2.wad
   --------------------------------------------------------------------
   Freedoom — ein vollständig freier Ersatz für die (NICHT freien)
   Original-Spieldaten von DOOM (eigene Level, Grafiken, Sounds, Musik).

   Lizenz : BSD-3-Clause (siehe freedoom-COPYING.txt)
   Quelle : https://freedoom.github.io/  (Version 0.13.0)

WICHTIG — was NICHT enthalten ist:
   Die kommerziellen Original-WADs von id Software (DOOM.WAD, DOOM2.WAD)
   sind urheberrechtlich geschützt und dürfen NICHT bereitgestellt
   werden. CrazyFamily Doom nutzt ausschließlich Freedoom.

"CrazyFamily" ist die Marke/Eigenleistung von Kevin Herrmann; das
Branding (Menü, Logo, Layout) ist eigenständig.
